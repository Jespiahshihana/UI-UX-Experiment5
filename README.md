# UI-UX-Experiment5

## Aim:
The UX Design Experiment aims to improve user experience in two digital platforms:

Fashion E-commerce Website – Hanabana Clothing Brand

The goal is to design a visually appealing and user-friendly fashion website that allows customers to easily browse clothing collections, view product details, and purchase items online. The platform should provide a smooth shopping experience, clear navigation, and attractive product presentation to encourage customers to explore and buy products.

Healthcare E-commerce Website

The goal is to design a healthcare product website that allows users to easily find and purchase wellness and health-related products. The platform should focus on simplicity, trust, and accessibility so users can quickly locate products, understand their benefits, and make secure purchases.

## Algorithm:
Identify user pain points through feedback, observation, and basic usability analysis of existing e-commerce websites.

Define UX problem statements based on common issues users face while shopping online.

Choose suitable research methods:

  Hanabana Fashion Website

  User observation while browsing clothing sites

  Surveys about online shopping preferences

  Usability testing for navigation and product browsing

  Healthcare E-commerce Website

  Surveys to understand health product purchasing behavior

  Heuristic evaluation of existing healthcare websites

  User feedback on product search and trust factors

Create user personas representing typical users such as fashion shoppers and health-conscious buyers.

Use insights to design improvements focusing on usability, visual appeal, product discovery, and user trust.
